# Regolith Red 🔴

A retro-inspired turn-based strategy game set on Mars. 
Command your troops, claim regions across the red planet, and outlast your opponent.

## Gameplay
- **Left click** a region to select and capture it
- **Right click + drag** to rotate the camera around the planet
- Take turns deploying troops to expand your control across Mars

## Features
- LAN and online multiplayer
- 3D globe with custom sphere-to-UV coordinate mapping for region selection
- Classic retro aesthetic via screen-space color quantization and resolution downscaling
- Depth and normal-based edge detection for stylized outlines with light-aware shading
